# Spaceships party
An inspiration from astro party game

### Requirements
- If you are not using Linux, remove the `SLD2/` prefix of the import
- SDL2: 2.30.6
- SDL2_image: 2.8.4
- SDL2_ttf: 2.24.0

### How to run
- `make all` to build, output at `dist/`
- `make run` to run the built executable
